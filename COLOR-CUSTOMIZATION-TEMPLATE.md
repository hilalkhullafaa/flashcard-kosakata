## RINGKASAN KATEGORI WARNA

### Total Elemen yang Perlu Warna:

1. **Background** (11 elemen)
   - Body, Header, Footer
   - Cards, Modals, Sections
   - Gradients

2. **Buttons** (4 jenis × 4 states = 16 warna)
   - Primary (Blue)
   - Success/Game (Yellow)
   - Danger (Red)
   - Secondary (Gray)
   - Masing-masing: Light Base, Light Hover, Dark Base, Dark Hover

3. **Text** (5 jenis × 2 modes = 10 warna)
   - Primary, Secondary, Muted, Accent, Flashcard

4. **Borders** (4 jenis × 2 modes = 8 warna)
   - Card, Input, Focus, Flashcard

5. **Badges & Indicators** (5 jenis × 2 modes = 10 warna)
   - Chapter, Checkmark, Number, Progress Fill, Progress BG

6. **Hover States** (2 jenis × 2 modes = 4 warna)
   - Card Hover, Source Hover

7. **Input Fields** (3 jenis × 2 modes = 6 warna)
   - Input BG, Input Text, Search BG

8. **Special Elements** (4 elemen)
   - Modal Overlay, Theme Toggle, Error BG, Error Text

1. **Isi semua kode warna** di atas dengan warna pilihan Anda
2. **Berikan file ini kepada saya** dengan format:
   ```

   ```
3. **Saya akan menerapkan** semua warna ke aplikasi
4. **Verifikasi hasil** di browser Anda

## TIPS MEMILIH WARNA

1. **Kontras**: Pastikan teks mudah dibaca di atas background
2. **Konsistensi**: Gunakan palet warna yang harmonis
3. **Accessibility**: Ikuti standar WCAG untuk kontras warna
4. **Dark Mode**: Warna dark mode harus lebih gelap dari light mode
5. **Hover States**: Warna hover harus sedikit lebih gelap/terang dari base
